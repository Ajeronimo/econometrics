Esta serie surge a partir de la necesidad de probar distintas técnicas de modelado para representar fenomenos complejos como lo es la estinación del ingreso laboral. Cada sección está pensada como una guía didactica que describe el proceso de utilización de los diferentes modelos teniendo la finalidad de identificar elementos practicos de política púbica basada en evidencia.
